Historia powstania

Java powstała w latach 90. w firmie Sun Microsystems. Jednym z głównych twórców był James Gosling. Początkowo język miał być używany w urządzeniach elektronicznych.

Filozofia projektowa

Najważniejszą zasadą Javy jest „Write Once, Run Anywhere”, czyli „napisz raz, uruchom wszędzie”. Oznacza to, że jeden program może działać na różnych systemach.

Przykład praktyczny

Ten sam program można uruchomić na Windowsie, Linuxie lub macOS, jeśli na komputerze jest zainstalowane środowisko Java.

Główna idea

Java miała być językiem prostym, przenośnym, bezpiecznym i wygodnym do tworzenia dużych aplikacji.

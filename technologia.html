Paradygmaty

Java jest językiem obiektowym. Kod organizuje się w klasy i obiekty, co pomaga tworzyć większe i bardziej uporządkowane programy.

Typowanie

Java ma typowanie statyczne. Typ zmiennej trzeba podać przed użyciem.

int liczba = 10;
String tekst = "Java";

Zarządzanie pamięcią

Java posiada Garbage Collector, który automatycznie usuwa niepotrzebne obiekty z pamięci.

JVM

Kod Java jest kompilowany do kodu bajtowego, a potem uruchamiany przez maszynę wirtualną JVM.

Obsługa błędów

Błędy można obsługiwać za pomocą bloków try-catch.

try {
    int wynik = 10 / 0;
} catch (Exception e) {
    System.out.println("Wystąpił błąd");
}

Bezpieczeństwo

Java nie daje bezpośredniego dostępu do wskaźników pamięci, dzięki czemu zmniejsza ryzyko wielu błędów.
